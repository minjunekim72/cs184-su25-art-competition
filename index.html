D

Caption

"Expensive hardware is insignificant next to the power of an optimized rasterizer" - Darth Shader

How is the artwork created?

I wrote a Python script that generated an SVG file. First I looked through the given test SVG images to see the tags that would work with the rasterizer: I relied on "rect" and "polygon". The setup that consists of a background and a green lightsaber is hard-coded (needed to spend some time figuring out the SVG attributes). Death Star is implemented with a for loop that uses sin and cos functions as the angle of traversal increases. The stars have different colors and are positioned randomly using Python's "random" module.

How is the artwork created?

The following work of mine is based on the idea that color space would exist physically in reality. In the work, the color space is shown as a cube in RGB color space. You can pick colors by touching the cube, like a painter's palette. I would call it "PaletteCube".
